What is TypeScript?

TypeScript is essentially an addition to vanilla JavaScript that adds typing! Usually, JavaScript is loosely typed, which means it can be difficult to see what data is being passed around. Especially in something like web development, adding typing can be extremely useful because there are so many moving pieces between the frontend and backend. Errors will be much more useful, and development will be faster with the use of TypeScript. Another big benefit of using TypeScript is its scalability. If you're expanding a website and can't wrap your head around what is actually happening, it'll be much harder to maintain.
